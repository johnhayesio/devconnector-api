# DevConnector API

> Backend API for DevConnector application, which is a developer social network.

## Usage

Rename "config/production.json" to "config/default.json" and update the values/settings to
your own.

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run server

# Run in prod mode
npm start
```
